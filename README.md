# Automatic License Plate Recognition (ALPR)

## Overview
This project implements an ALPR system for Malaysian vehicle plates.
A baseline system is implemented and an improved innovation version
is proposed to enhance recognition performance.

## Project Structure
- baseline/ : Baseline ALPR implementation
- innovation/ : Improved ALPR implementation
- models/ : Trained model
- dataset/ : Dataset description and samples
- results/ : Output results for comparison
